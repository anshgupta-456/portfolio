## 1. GSAP Animations
- Scroll-synced horizontal project showcase

- Responsive animation logic using `matchMedia()`  

## 2. Fully Responsive
- Desktop: GSAP pinned horizontal scrolling

- Touch-friendly design  
- Adaptive layouts and typography  

## 3. 3D Components
- Tech stack section built with Three.js
